# 无重复字符的最长子串
## 给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。
```python
    class Solution:
    def lengthOfLongestSubstring(self,s:str)->int:
        if len(s) <=1:
            return len(s)
        s_dist={}
        temp=''
        for char_s in s:
            if char_s not in temp:
                temp += char_s
            else:
                s_dist[len(temp)] = temp
                index = s.find(char_s)
                temp += char_s
                temp = temp[index+1:]
        return max(s_dist)

```
