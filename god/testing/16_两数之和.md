## 两数之和
给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。
[2,7,11,15]


```python
class Solution:
    def twoSum(self, nums, target: int):
        if len(nums)<=1:
            return nums
        i = 0
        while i <len(nums):
            if nums[i]<= target:
                last = target-nums[i]
                for j in range(i+1,len(nums)):
                    if last == nums[j]:
                        return i,j
                i+=1
            else:
                i+=1
if __name__ == '__main__':
    sum = [3,2,4]
    A=Solution()
    print(A.twoSum(sum,9))
```
