| 类型      | 字节         |
| --------- | ------------ |
| `byte`    | 1            |
| `char`    | 2            |
| `boolean` | 没有具体规定 |
| `short`   | 2            |
| `int`     | 4            |
| `long`    | 8            |
| `float`   | 4            |
| `double`  | 8            |
