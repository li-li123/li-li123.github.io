# 用户输入和while循环
## 函数input（）的工作原理 
`eval()` 函数在 Python 中用于执行字符串表达式，并返回表达式的值。
```python
number = int(input("Enter a number，and I'll tell you if it's even or odd"))
# number = int(number)
print(number)
print(type(number))
```
###  使用int（）来获取数值输入
使用input()时，Python将用户输入解读为字符串。
```python
age = input("How old are you?")
print(age)
```
### 求模运算符
求余运算符：%
```python
a = 4%5
b = 6%5
print(a)
print(b)
```
